Lifecycle

interface Lifecycle(source)

A holder of Lifecycle.State that can be observed for changes.

Possible transitions:

[INITIALIZED] ──┐(1)
                ↓
     (6)┌── [CREATED] ────┐(2)
        ↓       ↑ (5)     ↓
   [DESTROYED]  └──── [STARTED] ──┐(3)
                          ↑       ↓
                       (4)└── [RESUMED]

(1): ON_CREATE
(2): ON_START
(3): ON_RESUME
(4): ON_PAUSE
(5): ON_STOP
(6): ON_DESTROY

fun Lifecycle.currentStateAsState(): State<Lifecycle.State>
